# 이하민 - 풀스택 엔지니어

## 전문 분야 요약
웹·온디바이스 LLM·실시간 스트리밍·로보틱스를 '출시 가능한 제품'으로 잇는 풀스택 엔지니어.
모바일 온디바이스 AI와 실시간 미디어/로보틱스를 잇는 브릿지·스트리밍·저지연 제어 전문성 보유.
프라이버시 중심 아키텍처와 PWA/엣지 배포로 사용자 경험과 성능을 동시에 달성.
제품 초기화부터 배포/운영 자동화까지 엔드투엔드 오너십으로 다수 프로젝트 출시 및 운영 경험 보유.

## 핵심 역량
- 풀스택 개발: React, Next.js, Vue, Nuxt.js, Svelte Kit, TypeScript, JavaScript
- 백엔드: Node.js, Nest.js, Express.js, Prisma ORM, Drizzle ORM
- 데이터베이스: PostgreSQL, MySQL, Redis, Supabase
- 모바일: React Native, Expo, 온디바이스 AI (ONNX Runtime)
- 실시간 통신: WebRTC, WebSocket, UDP 프로토콜
- 로보틱스: 로봇 제어 시스템, SLAM 매핑, 경로 계획
- AI/ML: 온디바이스 LLM, STT/TTS, 이미지 처리 (OpenCV)
- 클라우드 & DevOps: AWS, Vercel, Cloudflare, Docker, Railway
- 도구: Git, Electron.js, Web Bluetooth, PWA

- AI 기반 대시보드 개발 및 Supabase 기반 DB 연동 구성
